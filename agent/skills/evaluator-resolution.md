# Evaluator Resolution

Use this skill when top submissions are close or when the buyer disputes the
automated result.

Rules:
- A close race is less than 8 points between first and second place.
- Do not release escrow automatically in a close race.
- Ask evaluators to compare only the evidence and acceptance criteria.
- Prefer short rationale tied to concrete checks.

Output:
- vote question
- finalists
- evidence delta
- recommended evaluator rationale format
